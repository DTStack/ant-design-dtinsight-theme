通过鼠标或键盘，输入范围内的数值。

## 设计规范

- 默认宽度 100px，边框颜色 #D8DAE2，4px 圆角；
- hover 状态边框颜色 #1D78FF；
- 点击状态边框颜色 #005CE6；
- 前后缀单位背景 #F9F9FA，边框 #D8DAE2，左右边距 16px，字体 12px/#3D446E 居中，默认宽度 208px；

## 何时使用

当需要获取标准数值时。

## API

|类名  |说明  |使用示例  |
|---------|---------|---------|
|.dt-input-number-disabled  | 禁用状态的颜色   | 禁用状态  |
