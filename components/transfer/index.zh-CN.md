双栏穿梭选择框。

## 如何使用

- 需要在多个可选项中进行多选时。

- 比起 Select 和 TreeSelect，穿梭框占据更大的空间，可以展示可选项的更多信息。

## API

|类名  |说明  |使用示例  |
|---------|---------|---------|
| .dt-transfer-without-header-count | 隐藏 header 中的下拉操作及数量，调整 title 位置 | -
